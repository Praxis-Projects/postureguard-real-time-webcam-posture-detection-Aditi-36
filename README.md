# Internship Project: Pose Landmark Tracker

A Python-based project developed during my internship focused on computer vision, real-time tracking, and data extraction. The application utilizes computer vision frameworks to capture and refine landmark coordinates from a live camera feed or video input.

## 🚀 Features
* **Real-Time Tracking:** Captures landmark coordinates with low latency.
* **Refined Data Output:** Cleaned up and formatted print statements/logs for tracking coordinate points precisely.
* **Modular Architecture:** Structured with a dedicated source directory (`src/`) for scalable development.
